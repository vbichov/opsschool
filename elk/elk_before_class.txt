Logstash
========

1. download java 8 (latest version) from oracle 
  https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

2. extract and set JAVA_HOME env var
  export JAVA_HOME=<path to java>/jdk1.8.0_<latest>

3. download logstash latest version:
  https://www.elastic.co/downloads

4. create conf file test.conf
using stdin and stdout plugins:
https://www.elastic.co/guide/en/logstash/current/plugins-inputs-stdin.html
https://www.elastic.co/guide/en/logstash/current/plugins-outputs-stdout.html

```
input { 
  stdin { } 
}
filter {
}
output {
  stdout { codec => json }
}
```

5. run logstash - try sending it some random message
./bin/logstash -f ./1test.conf 

6. try adding a kv filter 
https://www.elastic.co/guide/en/logstash/current/plugins-filters-kv.html
and sending the following message ip=1.2.3.4 error=REFUSED and see what happens
What happens when `codec => json` is removed? 
What is the default codec?

7. go over https://www.elastic.co/guide/en/logstash/current/getting-started-with-logstash.html
